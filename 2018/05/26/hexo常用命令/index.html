[hexo常用命令]

发表于 2018-05-26 | 分类于 hexo常用命令

参考文档:http://theme-next.iissnan.com/getting-started.html

简书参考文档:https://www.jianshu.com/p/39562a0d8eb6?utm_campaign=maleskine&utm_content=note&utm_medium=seo_notes&utm_source=recommendation

博客根目录的 _config.yml 为 “站点配置文件”，包括 SEO、主题、布局、插件等配置项

Site

title: xxx的博客
subtitle: 副标题
description: 对站点的描述
keywords: 关键词
author: 作者
language: 语言（中文简体为：zh-Hans）
timezone: 时区（国内这里填写：Asia/Shanghai）

常用命令：
使用npm安装Hexo：
npm install hexo-cli -g

查看Hexo版本：
hexo version

本地文件提交到GitHub

// 删除旧的public文件
hexo clean
//生成新的public文件

hexo generate
或者
hexo g

//开始部署
hexo deploye
或者
hexo d

文章发布：
hexo new “文章标题”
文章建好了执行以下命令
本地预览
hexo clean
hexo s

发布到GitHub
hexo d -g
或者
hexo g && hexo d

新建一个自定义页面
hexo new page folder

生成分类，标签，关于我，首先在主题配置文件中开启相关内容
利用 hexo new page folder 创建相关的文件夹和文件编写相关的内容就可以了
